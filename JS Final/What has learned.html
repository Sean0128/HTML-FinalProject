First Week

這兩周所學到的語法整理:
alert():
　　當程式執行到aler()的時候會彈出一個提示的視窗，視窗裡的內容為寫在aler()括弧裡的東西，語法:alert("請輸入內容")

comfirm():
　　與alert()相似，但需要使用者點擊確認或取消，多會搭配if條件判斷式使用。如右圖，螢幕上會創建一個上面寫著Try it 的按鈕，當按下確認後p標籤會執行script裡面的程式，如果按確認，txt就會等於You pressed ok!。

script:
　　script 內的東西要用id來導入，在script的最後加入document.getElementById("想要命名的id") = "function裡面設定的變數"，然後在想要使用script的標籤後面加入 = "想要命名的id"，就能夠讓標籤內的東西照著script做。參考圖(一)

function():
　　function 類似python的def，語法為：function FunctionName() {}，大括弧內就是function的主要內容，而function的小括弧內可以設定一個變數，ex：function FunctionName(y){}，然後可以再將javascript:FunctionName("number")放進a標籤改變變數的值，參考圖(二)
